GainLens AI – AI Fitness Tracker

GainLens AI is an AI-powered fitness tracking web application that uses computer vision and pose estimation to analyze workouts and improve exercise tracking. The project combines a modern React frontend with a Flask + MediaPipe backend to provide an interactive fitness experience with real-time pose analysis and workout monitoring.

Live Website https://ai-fitness-tracker-ashy.vercel.app
